3) REGRA DE PONTUAÇÃO (CRÍTICO — implemente exatamente assim)
--------------------------------------------------------------------------
Dado o resultado real (RC x RF) e o palpite (PC x PF), por jogo, vale
SEMPRE A MAIOR pontuação aplicável:

  25 pts — Placar exato:  PC==RC E PF==RF.
  18 pts — Acertou o vencedor E a quantidade de gols do VENCEDOR.
  15 pts — Acertou o vencedor E a quantidade de gols do PERDEDOR.
  12 pts — Acertou o vencedor E o SALDO de gols.
  10 pts — Acertou apenas o vencedor.
   5 pts — Acertou que seria EMPATE, sem acertar o placar exato.
   0 pts — Errou o resultado (ou faltou resultado/palpite -> sem pontuação).

PSEUDOCÓDIGO DE REFERÊNCIA (já validado, use igual):

  function points(RC, RF, PC, PF):
    if (qualquer um for null/vazio) return null   // jogo sem resultado ou sem palpite
    if (PC==RC && PF==RF) return 25
    if (RC==RF)                                    // resultado foi empate
        return (PC==PF) ? 5 : 0
    venceuCerto = (RC>RF && PC>PF) || (RC<RF && PC<PF)
    if (!venceuCerto) return 0
    p = 10
    if (max(RC,RF) == max(PC,PF)) p = max(p,18)    // gols do vencedor
    if (min(RC,RF) == min(PC,PF)) p = max(p,15)    // gols do perdedor
    if ((RC-RF) == (PC-PF))       p = max(p,12)    // saldo
    return p

CASOS DE TESTE OBRIGATÓRIOS (devem passar):
  points(2,0, 2,0) == 25     points(2,1, 1,0) == 12
  points(1,1, 0,0) == 5      points(4,1, 2,1) == 15
  points(2,0, 2,1) == 18     points(2,1, 0,2) == 0
  points(4,1, 3,0) == 12     points(1,1, 2,2) == 5
  points(2,0, 1,0) == 15     points(0,3, 1,3) == 18

RANKING:
  - Pontos do participante = soma de points() sobre todos os jogos onde há
    resultado E palpite.
  - "Placares exatos" = quantos jogos deram 25.
  - Ordenar por: pontos (desc) -> placares exatos (desc) -> nome (asc).
  - DESEMPATE OFICIAL: mais placares exatos.
